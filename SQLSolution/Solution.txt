	The database name is log_analyzer while the table name is logs
	
(1) Write MySQL query to find IPs that mode more than a certain number of requests for a given time period.

    Ex: Write SQL to find IPs that made more than 100 requests starting from 2017-01-01.13:00:00 to 2017-01-01.14:00:00.

	ANSWER:
	
	SELECT  ip, COUNT(ip) as requests
	FROM logs
	WHERE 
	date >= '2017-01-01 13:00:00' and date <= '2017-01-01 14:00:00' 
	GROUP BY ip
	HAVING requests > 100;

(2) Write MySQL query to find requests made by a given IP.

	ANSWER
	
	The questions says requests made by a given IP, this means an IP will be provided to run the query.
	
	SELECT * FROM logs WHERE ip = '192.168.169.194';
